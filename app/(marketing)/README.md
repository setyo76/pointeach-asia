# Marketing (Landing) — Struktur & Konvensi

Folder `app/(marketing)` berisi semua halaman marketing / landing yang akan tampil di root domain.

Konvensi:

- `layout.tsx` — layout global untuk halaman marketing (Navbar + Footer).
- `page.tsx` — halaman landing utama (`/`) — hero, fitur, pricing, FAQ.
- `features/page.tsx` — halaman fitur (`/features`).
- `pricing/page.tsx` — halaman harga (`/pricing`).
- `faq/page.tsx` — halaman bantuan/FAQ (`/faq`).

Catatan:

- Folder ini adalah route group `(marketing)`, sehingga nama folder tidak muncul di URL.
- Taruh komponen spesifik landing di `components/landing/`.
- Untuk menambahkan halaman marketing baru, buat folder baru di dalam `(marketing)` dan tambahkan `page.tsx`.

Contoh: `app/(marketing)/contact/page.tsx` akan tersedia di `/contact`.

Saran pengembangan:

- Simpan data statis (fitur, testimonial, pricing) di `constants/landingData.ts`.
- Gunakan komponen kecil yang dapat dipakai ulang di `components/landing/`.
